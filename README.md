# Mini Calculadora em Python

Este é o meu primeiro projeto em python.
Uma mini calculadora que funciona através de um menu.


## Para que serve

1. Calcular o triplo de um número
2. Calcular o quadrado de um número
3. Somar o triplo com o quadrado
0. Encerrar o programa

## Requisitos

Python 3.10+

## Como executar

1. Abra o terminal na pasta do arquivo
2. Execute esseb comando:
```bash
    Python main.py
